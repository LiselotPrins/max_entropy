Project for Bachelor's thesis 'Cosmological field analysis with the principle of maximum entropy'
Liselot Prins
13-06-2025

Python version: 3.12.4
'requirements.txt' contains all packages necessary to run the project.

Various jupyter notebooks are included which generated the results for the
thesis.
The folder 'python_files' contains .py-files which are used in the notebooks.
The folder 'mathematica_notebooks' contains the files used for solving the 
system of equations in Section 3.1.
The folder 'figs' holds all figures used in the thesis.

The data is not included in this project.
